# deutschocean

**Status:** MVP / Portal-Projekt  
**Demo:** https://deutschocean-vylu.vercel.app/app

Content- und Portal-App mit React, Firebase, Auth, Dashboard-Strukturen und Tiptap-Editor.

## Zweck

Das Projekt bündelt typische Portal- und Content-Management-Funktionen:

- Authentifizierung und geschützte Bereiche
- Dashboard-Strukturen
- redaktionelle Inhalte
- Rich-Text-/Editor-Flows mit Tiptap
- Firebase als Backend
- React Router für App-Navigation

## Tech Stack

- React 18
- Vite
- Firebase
- Tiptap Editor
- React Router
- Vitest
