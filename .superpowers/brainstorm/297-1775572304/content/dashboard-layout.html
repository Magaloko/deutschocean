Wo landen die Mathe-Spiele?

Sollen Mathe-Spiele in die bestehenden Level-Sektionen gemischt werden, oder bekommt Mathe einen eigenen Bereich?

A

Gemischt — einfach hinzufügen

Mathe-Spiele kommen einfach zu den bestehenden Sektionen dazu (Kindergarten / Volksschule / NMS), genau wie FrüchteZählen. Dashboard bleibt wie es ist.

Pro

  • Kein Umbau nötig
  • Schnell umsetzbar
  • Ein Ort für alles

Contra

  • Dashboard wird sehr lang
  • Kein klarer Deutsch/Mathe-Fokus
B

Tabs — Deutsch / Mathe

Dashboard bekommt zwei Tabs: Deutsch und Mathe. Jedes Fach hat seine eigenen Spiele und Level-Sektionen.

Pro

  • Klare Trennung
  • Skaliert für mehr Fächer später
  • Kinder können Fokus wählen

Contra

  • Dashboard-Umbau nötig
  • Mehr Aufwand
C

Eigene Seite — /mathe

Mathe bekommt eine komplett eigene Seite (wie /app/mathe), mit eigenem Dashboard und eigenen Sektionen. Navigation erhält einen Mathe-Link.

Pro

  • Maximal skalierbar
  • Kann unabhängig wachsen

Contra

  • Größter Umbau
  • Navigation wird komplexer

Meine Empfehlung: B — Tabs, weil 7 Mathe-Spiele das Dashboard sonst zu lang machen.
